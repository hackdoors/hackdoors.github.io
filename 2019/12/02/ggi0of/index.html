函数

1.使用 def 定义，
后面紧跟着函数名和参数列表，参数没有指定类型，并且也不需要指定返回值的类型

2.接收两个数，使用 return 返回较大的值

3.一个没有返回值的函数

没有指定返回值的函数，默认会返回 None
_
4.函数的默认参数，使用和 C++ 相同
参数的设定必须从右向左

5.调用函数时指定某一个参数的值

6.给函数传入不确定个数的参数之元组(*)
如果参数添加了 * 表示接受的数据会被组合成一个元组进行保存，传入的数据是不能够被改变的

7.给函数传入不确定个数的参数之字典(**)
当参数指定为 ** 的时候，表示传入的会被组织成一个字典，
传入的必须是键值对传入的内容能够被修改

8.一般在使用的过程中，两种方式会混用

9.Python 中的作用域分为全局的和函数内(局部的)

8.全局的访问不到局部的
