模块

模块中写全局变量

_使用 import 可以直接导入模块，导入模块的同时会执行其中的代码，但是同一个模块的代码不论直接 import 多少次，都只执行一次, _


拓展：
import ..　　　　 　　　　#导入整个模块
from .. import ..　　　　　 #导入模块中的类、函数或者变量
from .. import * 　　　　　#导入模块中的所有公开成员
from .. import .. as ..　　 #这个是给导入的成员取个别名
